This correctness issue must be fixed before the change can be accepted.

- The function should first capture the user ID, then safely iterate over a copy of the session keys (or build a list of sessions to delete) and delete them without referencing the already‑removed entry.
- Ensure that the global `sessions` mapping is correctly defined and thread‑safe if used in a concurrent environment.
