## 3. Integration Tests

|------|----------|-----------------|
| **Happy Path 1** | User `alice` authenticates, receives token, accesses `/app/protected` on App A | 200 OK, payload contains user data |
| **Happy Path 2** | Same token used to access `/app/protected` on App B | 200 OK, same payload |
| **Happy Path 3** | Token revoked via `/revoke`, then used on App A | 401 Unauthorized, `{ "error": "revoked_token" }` |
| **Edge 1** | Submit 101 auth requests in 60 s from same IP | 429 Too Many Requests, `{ "error": "rate_limit_exceeded" }` |
| **Edge 2** | Use malformed JWT (invalid signature) | 401 Unauthorized, `{ "error": "invalid_token" }` |
| **Edge 3** | Token with `exp` in past | 401 Unauthorized, `{ "error": "token_expired" }` |
